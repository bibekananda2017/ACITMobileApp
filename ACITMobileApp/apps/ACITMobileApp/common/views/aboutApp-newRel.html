What's New in this Latest Version 2.1.1

  • Message Board implemented and deployed as a minimal viable feature within a week, for use by a target set of users. This feature provides a message board for group collaboration amongst users of this version of the app.
  • 'About this app'- gives a background and story of the development of this APP, an overview of the platform, tech stack and Key features of the app.
  • Personalized content notification (Push Service) - As part of personalized content notification, we are using Bluemix-Push Service subscription approach. Where the user subscribes to certain tags (categories) depending on their interest. While broadcasting a message the administrator chooses the tag (category) which the message belongs and Bluemix-Push Service sends the message based on subscription of the tag.
