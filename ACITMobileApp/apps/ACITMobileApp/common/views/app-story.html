The ACIT App Journey

Objective

  • Enable easy access to information and features for ACIT team members
  • Create a more personalized interaction with ACIT for users across Accenture
  • Leverage and showcase key features of IBM MobileFirst platform

Journey

  • ACIT dev team started with no mobile development experience, and were able to upskill and pick up the required technology and leverage IBM MobileFirst platform, to design and develop this application.
  • Core development team and leads based in ACIT India center, including IDC UI/UX SME's; Guidance and inputs from ACIT Oslo; Key stakeholders in ACIT North America.
  • Collaboration across ACIT centers, developing this application in an agile manner, with features developed and enhanced across multiple sprints and releases.
  • Hybrid application - Mobile web and native; Hybrid cloud: with components across public cloud (IBM Bluemix PaaS), private cloud (IBM Softlayer IaaS dedicated for ACIT), private on-premise (leveraging backend services within Accenture enterprise)
